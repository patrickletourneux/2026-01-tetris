# Tetris React/Redux TypeScript

Testing Claude Code by building a Tetris game with React, Redux and TypeScript.
And learn architecture haxagonale in the same time.

## Tech Stack

- React 19
- Redux Toolkit
- TypeScript
- Vite
- Jest (testing)

## Getting Started

```bash
cd tetris-hexagonal
npm install
npm run dev
```

The game will be available at http://localhost:5173/

## Controls

- **Left/Right arrows**: Move piece
- **Down arrow**: Speed up descent
- **Up arrow / Space**: Rotate
- **Enter**: Instant drop
- **P / Escape**: Pause

## Tests

```bash
cd tetris-hexagonal
npm test
```
